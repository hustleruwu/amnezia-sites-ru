# amnezia-sites-ru
В этом json файле хранятся домены и IP-адреса множества российских сайтов(также пытался найти и зеркала), для того чтобы добавить их в обход раздельного тунеллирования AmneziaVPN.

1. Скачайте json себе на устройство
2. В настройках AmneziaVPN включите раздельное туннелирование
3. Выберите режим "Адреса из списка не должны открываться через VPN"
4. Импортируйте список через три точки внизу
